# Netflix_Clone_using_emergent
#  Netflix Clone

A fully responsive **Netflix Clone** built using **Emergent**, replicating the core UI and user experience of Netflix.  
This project focuses on clean design, smooth navigation, and modern web development practices.

---

##  Live Demo
(https://clonecraft-1.preview.emergentagent.com/)

---

##  Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Framework / Platform:** Emergent  
- **UI Design:** Netflix-inspired modern layout  
- **Version Control:** Git & GitHub  

---

##  Features
-  Netflix-style homepage UI  
-  Fully responsive design (mobile + desktop)  
-  Movie/Show cards layout  
-  Clean and intuitive user interface  
-  Fast loading and smooth interactions
